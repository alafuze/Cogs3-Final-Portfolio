Projects



studyBuddy

studyBuddy is a webpage designed to help college students find peers within their field of academia to prepare for exams and study for their classes. This site is meant to match students together where they will not only connect with their peers in school, but create lasting friendships!

This app was designed using HTML and CSS using a Bootstrap framework in order to construct the most functional styling for this webpage's design.

The components of this webpage include a pricing plan that accomodates to all students' budgets, student testimonials and press reviews from some of the most prestigious schools in the U.S., and mulitple sources for easy download.





myCLOCK


myCLOCK is an app that was designed to track your sleep and monitor your circadian rhythms in order to calculate a sleep schedule that is right for YOU.

This app is meant to log the typical hours the user tends to go to sleep at night and wake up the next morning. If the user wants to adjust their sleep schedule, then the app can calculate what time the user should go to bed in order to get their preferred amount of sleep. myCLOCK will then design customs alarms that will wake the user up in the morning in order to keep their circadian rhythms on schedule.

To design this app, I used Figma which allowed me to customize all app icons and create a functional layout to optimize the user experience.

I used photo galleries to make the app more interactive, which then open into an online forum where users can interact with each other and form communities surrounding their internal clocks.

With the use of Figma, I have the capabilities to design any app for all purposes in a creative design that will send your app straight to the top of the charts.





Dice Game


The Dice Game is a webpage that was designed to be interactive with the user and keep track of the total amount of money won. The user begins the game by clicking "Let's Play!" and then typing the amount of rolls that they want to play. If the dice read 9 or above, you win $5. Roll 5 or below, you lose $5. In between is considered a draw.

I designed this game using a combination of JavaScript, HTML and CSS. I was able to code the game to store an ongoing total so the user can continue to play with the same pot of money.

With my skills in JavaScript, I can easily code and design games to be enjoyed by all ages!
