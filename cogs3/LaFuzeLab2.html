How to Define a Table

<tr>
Used when you want to define a row in a table
<th>
Used when you want to define a header in a table
<td>
Used when you want to add data to a cell within a table
<thead>
Used when you want to group content under specific headers in a table
<tbody>
Used when you want to group body content within a table
<tfoot>
Used when you want to group footer content within a table
<caption>
Used to you want to define the caption of a table
Colspan
Used when you want to expand the size of a column wihtin a table
Rowspan
Used when you want to expand the size of a row within a table
Cell Spacing
The space between each cell
Padding
The space between the edges of the cell and the cell's content
Table Borders
CSS is used to define table borders by their size, color, and style
example: table{ border: 1px solid black;}
Border Width
CSS controls border width by defining the amount of pixels in the border (1px, 5px, etc.)
